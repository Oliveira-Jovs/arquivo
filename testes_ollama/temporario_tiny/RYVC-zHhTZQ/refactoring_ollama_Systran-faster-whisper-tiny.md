As pessoas que trabalham em uma equipe de desenvolvimento sofrem com a conferência, pois têm que integrar e juntar as contribuições feitas individualmente durante a fase de construção do sistema. No momento, fazem isso através de ferramentas comuns como o Git Hub, Git Lab e Git RPG, que analisam conflitos quando as pessoas tentam mudar a mesma parte do código do sistema.

Por exemplo, quando duas pessoas tentam alterar linhas consecutivas de um mesmo programa, método ou classe. No entanto, muitas vezes essas mudanças feitas individualmente pelas pessoas, apesar de serem conflitantes, não são detectadas por essas ferramentas, que infelizmente são limitadas e passam batidas, integrando as contribuições de forma equivocada. Isso pode levar a falhas no sistema quando este é utilizado pelos usuários, trazendo recursos negativos para muita gente.

Muitas vezes, esses problemas podem ser sérios, e para evitar essas limitações das ferramentas atuais de integração de código, nós desenvolvemos uma solução em colaboração com pesquisadores dos Estados Unidos. Pesquisamos e desenvolvemos ferramentas complementares que podem ser acopladas ao Git Hub para detectar esses problemas e conflitos, evitando que eles cheguem aos usuários finais.

A gente faz isso usando a combinação de técnicas de análise estática e dinâmica de código, que consegue detectar os chamados conflitos semânticos. Esses conflitos são causados quando pessoas diferentes de uma mesma equipe trabalham em partes diferentes do código, mas essas partes são dependentes e precisam ser ajustadas antes de serem integradas. No entanto, as pessoas individualmente não têm conhecimento disso.

As nossas ferramentas vão justamente nessa direção, sinalizando a necessidade desses ajustes para que o defeito possa ser antecipado e a falha do sistema evitada. Isso é especialmente importante porque esse tipo de defeito, causado pela interferência entre mudanças feitas por duas pessoas, é muito mais difícil de ser detectado e corrigido do que os defeitos individuais causados por apenas uma pessoa.
